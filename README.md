今日の晩ごはん決める君（改）

## ②課題内容（どんな作品か）
- 「今日のごはん、何にしよう…」という日々の精神的負担と献立作りのストレスを和らげる、
晩ごはん特化型のメニュー選定Webアプリです。
家族構成やその日の気力（HP）に合わせるだけでなく、「曜日ごとのライフスタイル」に
寄り添った最適なメニューを裏側で自動選定して提案してくれます。

## ⑤工夫した点・こだわった点
- - **ライフスタイルに合わせた曜日別の3グループ判定**
  JavaScriptの `Dateオブジェクト` を活用し、裏側で「月・金」「火〜木」「土日」の
  3つの曜日グループを自動で判別する仕組みを実装しました。
- **月金リメイク・作り置きロジックと個別アドバイス**
  「月曜日・金曜日に多めに作っておけば、翌日以降の自分が楽になる」という自炊のハックをロジック化。
  さらに条件分岐（`if文`）を細分化し、月曜日には「平日の後半が楽になる応援」、金曜日には
  「土日に料理をしなくて済む労い」といった、曜日に完全にパーソナライズされたアドバイスを表示するといった
  工夫を凝らしました。
- **機能性**
  初回起動時に「一人暮らし」「大人二人」「家族（小学生以上/以下）」の4つの家族構成を選択し、2回目以降は自動で気力選択画面からスタートすることによって毎回選択しなくって良いようにしました
- **ハンバーガーメニュー**
  画面をスッキリさせるため、設定リセットや今後の拡張用メニューを右上のハンバーガーメニュー内へと集約しました。
